# 冒险岛Online - 自动刷怪脚本

## 📋 项目说明

本项目提供两种自动刷怪方式：
1. **按键记录式** - 录制键盘操作循环回放（已完成）
2. **图像识别式** - 基于屏幕识别的完全自动化（已完成）

---

## 方式一：按键记录式

### 文件说明

| 文件名 | 说明 |
|--------|------|
| `key_recorder.py` | 完整版，功能最全，有菜单界面 |
| `mxd_auto_simple.py` | 简化版，命令行快速操作 |

### 快速开始

```bash
python mxd_auto_simple.py
```

**操作流程：**
1. 按 `r` 开始录制
2. 3秒内切换到游戏窗口
3. 执行刷怪操作（左+攻击+右+攻击+技能...）
4. 按 `ESC` 停止录制
5. 按 `p` 开始自动刷怪
6. 按 `ESC` 停止

---

## 方式二：图像识别式 ⭐新

### 功能特性

| 功能 | 说明 | 状态 |
|------|------|------|
| 🎯 怪物检测 | 支持模板匹配和颜色检测两种模式，可切换 | ✅ |
| ⚔️ 自动战斗 | 支持近战/远程职业，自动移动和攻击 | ✅ |
| 🛡️ 异常处理 | 防卡死、被击退恢复、无怪探索 | ✅ |
| 🗺️ 地图适配 | 专为小地图优化 | ✅ |

### 文件说明

| 文件名 | 说明 |
|--------|------|
| `mxd_vision_auto.py` | 完整版，带调试显示，功能最全 |
| `mxd_vision_simple.py` | 简化版，快速启动 |

### 快速开始（简化版）

```bash
python mxd_vision_simple.py
```

**操作流程：**

1. **选择检测模式**
   ```
   检测模式 (c=颜色/t=模板): c
   ```
   - **颜色模式** (`c`): 自动识别红色系怪物，无需截图，即开即用
   - **模板模式** (`t`): 需要截图保存怪物图片，识别更精确

2. **选择职业类型**
   ```
   职业类型 (m=近战/r=远程): m
   ```
   - **近战** (`m`): 攻击距离80像素，会贴脸怪物
   - **远程** (`r`): 攻击距离250像素，保持距离攻击

3. **开始自动刷怪**
   ```
   命令 (s=开始/q=退出): s
   3秒后开始...
   ```

4. **停止**
   - 按 `ESC` 键停止

### 完整版使用

```bash
python mxd_vision_auto.py
```

完整版额外功能：

## ⚙️ 技术实现

### 系统架构

```
┌─────────────────────────────────────────────────────┐
│                   系统架构                          │
├─────────────────────────────────────────────────────┤
│  屏幕捕获  │  mss - 高性能截图（约15-30fps）        │
├─────────────────────────────────────────────────────┤
│  怪物检测  │  模板匹配: OpenCV matchTemplate        │
│            │  颜色检测: HSV颜色空间 + 轮廓检测       │
├─────────────────────────────────────────────────────┤
│  战斗策略  │  近战: 贴身攻击                        │
│            │  远程: 保持距离攻击                    │
├─────────────────────────────────────────────────────┤
│  异常处理  │  卡死检测、无怪探索、被击退恢复        │
├─────────────────────────────────────────────────────┤
│  操作执行  │  pynput - 键盘模拟                     │
└─────────────────────────────────────────────────────┘
```

### 怪物检测详解

#### 模式1: 模板匹配 (Template Matching)
- **原理**: 预先截取怪物图像作为模板，在屏幕中滑动匹配
- **优点**: 精确识别特定怪物，误识别率低
- **缺点**: 换个地图/怪物需要重新截图
- **适用**: 长期在同一地图刷同一种怪

#### 模式2: 颜色轮廓检测 (Color + Contour)
- **原理**: 检测特定颜色区域（如红色怪物），提取轮廓
- **优点**: 无需截图，通用性强，即开即用
- **缺点**: 可能有误识别（如红色技能特效）
- **适用**: 快速开始，怪物颜色明显的地图

## 🛡️ 防检测建议

冒险岛有Warden反作弊系统，请注意：

1. **操作随机性**
   - 每次移动/攻击都有 0.05~0.15s 的随机延迟
   - 攻击间隔不固定

2. **控制挂机时长**
   - 建议单次不超过 2-3 小时
   - 定期休息，模拟人工

3. **检测帧率**
   - 默认12-15fps，不追求过高频率
   - 降低CPU占用，减少异常

4. **使用建议**
   - 在小地图测试熟悉后再长期使用
   - 建议在离线地图/小号测试
   - 不要24小时不间断挂机

## ❓ 常见问题

**Q: 脚本需要管理员权限吗？**  
A: 如果游戏以管理员权限运行，脚本也需要，否则按键可能无法发送到游戏。

**Q: 图像识别模式识别不到怪物？**  
A: 尝试以下方法：
1. 切换到模板匹配模式，手动截图怪物
2. 调整颜色检测范围（修改代码中的HSV值）
3. 确保游戏窗口可见，没有被遮挡

**Q: 如何调整攻击距离？**  
A: 在完整版中可以设置，简化版默认为近战80px/远程250px。

**Q: 能识别所有怪物吗？**  
A: 颜色模式主要针对红色系怪物（如蜗牛、绿水灵）。其他颜色怪物需要：
1. 使用模板匹配模式
2. 或修改代码中的颜色范围

## 🔧 进阶配置

### 修改检测颜色

编辑 `mxd_vision_auto.py` 中的颜色范围：

```python
self.color_ranges = {
    "red": [(0, 100, 100), (10, 255, 255)],    # 红色怪物
    "blue": [(100, 100, 100), (130, 255, 255)], # 蓝色怪物
    # 添加更多颜色...
}
```

### 调整帧率

```python
self.target_fps = 15  # 降低可减少CPU占用
```

### 调整异常检测敏感度

```python
self.stuck_threshold = 20  # 卡死判定阈值（帧数）
self.no_action_threshold = 100  # 无怪判定阈值
```
