# 冒险岛Online - 自动刷怪脚本

## 项目说明

本项目提供两种自动刷怪方式：
1. **按键记录式** - 录制键盘操作循环回放
2. **图像识别式** - 基于屏幕识别的完全自动化

---

## 方式一：按键记录式

### 文件说明

| 文件名 | 说明 |
|--------|------|
| `key_recorder.py` | 完整版，功能最全，有菜单界面 |
| `mxd_auto_simple.py` | 简化版，命令行快速操作 |

### 快速开始

```bash
python mxd_auto_simple.py
```

**操作流程：**
1. 按 `r` 开始录制
2. 3秒内切换到游戏窗口
3. 执行刷怪操作（左+攻击+右+攻击+技能...）
4. 按 `ESC` 停止录制
5. 按 `p` 开始自动刷怪
6. 按 `ESC` 停止

## 快速开始（简化版）

```bash
python mxd_vision_simple.py
```

### 操作流程

#### 首次使用（新建配置）

1. **选择或新建配置**
   ```
   已保存的配置:
     [1] zhujiao

   请选择:
     编号: 使用已保存的配置
     n: 新建配置
     d: 删除配置
   ```

2. **选择截图区域**
   ```
   截图区域设置:
   1. 自动查找游戏窗口
   2. 鼠标拖拽选择区域
   3. 使用全屏截图
   ```
   - **自动查找**（推荐）: 通过窗口类名/标题自动匹配冒险岛窗口
   - **鼠标拖拽**: 截取全屏后用鼠标框选游戏区域
   - **全屏截图**: 使用主显示器全屏

3. **选择检测模式**
   ```
   检测模式 (c=颜色/t=模板): t
   ```
   - **颜色模式** (`c`): 检测特定颜色怪物，即开即用
   - **模板模式** (`t`): 框选怪物图像，识别更精确

4. **选择模板集（模板模式时）**
   ```
   已保存的模板集:
     [1] 280_3 (8个模板)
   请选择:
     编号: 使用已有模板集
     n: 新建模板集
   ```

5. **选择职业类型**
   ```
   职业类型 (m=近战/r=远程): r
   ```

6. **设置按键**
   ```
   跳跃键 (默认space): c
   攻击键 (默认x): v
   技能键 (逗号分隔, 默认a,s): a,s,d,f,t
   是否有位移技能? (y/n, 默认n): y
   位移技能键 (默认f): shift
   ```

7. **保存配置**
   ```
   是否保存配置? (y/n): y
   配置名称 (如: 神秘河_花蛇): zhujiao
   ```

8. **开始运行**
   ```
   命令 (s=开始/q=退出): s
   ```
   按 `ESC` 停止。

#### 再次使用（加载配置）

直接选择已保存的配置编号，所有设置自动恢复，无需重新配置。

## 技术实现

### 系统架构

```
+-----------------------------------------------------------+
|  ProfileManager  |  配置保存/加载，模板集管理               |
+-----------------------------------------------------------+
|  ScreenCapture   |  mss 高性能截图                         |
|                  |  EnumWindows 自动查找游戏窗口            |
|                  |  鼠标拖拽选择截图区域                    |
+-----------------------------------------------------------+
|  MonsterDetector |  模板匹配: OpenCV matchTemplate + NMS   |
|                  |  颜色检测: HSV + 轮廓检测 + 分水岭分割   |
+-----------------------------------------------------------+
|  CombatStrategy  |  高度分层决策：同层→下落穿台→上跳攻击   |
|                  |  BFS聚类找最密集怪物簇                   |
|                  |  keybd_event 模拟硬件输入                |
+-----------------------------------------------------------+
|  ExceptionHandler|  卡死检测（基于动作历史）                |
|                  |  无怪探索、被击退恢复                    |
+-----------------------------------------------------------+
```

### 按键输入方式

使用 `keybd_event` API 模拟硬件级键盘输入（DirectInput 可接收），而非 PostMessage：
- 自动计算 scan code
- 正确处理扩展键（方向键等）
- 支持 `try/finally` 保证按键释放，防止卡键

### 游戏窗口查找

通过 `EnumWindows` API 可靠查找冒险岛窗口：
1. 优先匹配窗口类名：`MapleStoryClass`、`MapleStory`
2. 其次匹配窗口标题：`MapleStory`、`冒险岛`、`新枫之谷`
3. 多个匹配时选择面积最大的窗口

### 怪物检测详解

#### 模板匹配 (Template Matching)
- 预先截取怪物图像，在屏幕中滑动匹配
- 支持 0.85x / 1.0x / 1.15x 三种缩放比例
- 自适应峰值间隔 + NMS 去重
- 模板以**模板集**形式管理，按文件夹组织

#### 颜色轮廓检测 (Color + Contour)
- HSV 颜色空间 + 轮廓检测
- 支持红/蓝/绿/黄四种颜色
- 距离变换 + 分水岭分割重叠怪物（完整版）
- 面积和宽高比过滤

### 战斗策略 - 高度分层决策

```
优先级1: 同层有怪 → BFS聚类找最密集簇 → 攻击或移向簇中心
优先级2: 下方有怪 → 下落穿台（下+跳）
优先级3: 上方有怪 → 上跳攻击（方向+上+二段跳+攻击）
```

撞墙检测：最近10次移动方向一致时自动换向。

### 配置系统 (ProfileManager)

- **配置文件**: 保存在 `configs/` 目录，JSON 格式
- **模板集**: 保存在 `templates/` 子目录中，每个模板集一个文件夹
- **保存内容**: 截图区域、检测模式、模板集、职业类型、按键、位移技能等
- **支持操作**: 保存、加载、删除配置和模板集

## 常见问题

**Q: 脚本需要管理员权限吗？**
A: 如果游戏以管理员权限运行，脚本也需要，否则按键可能无法发送到游戏。

**Q: 图像识别模式识别不到怪物？**
A: 尝试以下方法：
1. 切换到模板匹配模式，手动框选怪物
2. 调整颜色检测范围
3. 确保游戏窗口可见，没有被遮挡

**Q: 如何调整攻击距离？**
A: 新建配置时可自定义攻击距离。默认近战80px/远程250px。

**Q: 按键发不出去/卡键？**
A: 脚本已使用 `try/finally` 保证按键释放。如仍有问题，确保游戏窗口句柄正确绑定。

**Q: 如何切换不同角色/地图？**
A: 保存多个配置（如不同角色、不同地图），启动时选择对应配置即可一键切换。
