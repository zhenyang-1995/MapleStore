## 使用步骤详解

### 1. 配置管理

启动脚本后首先进入配置选择：

```
已保存的配置:
  [1] zhujiao

请选择:
  编号: 使用已保存的配置
  n: 新建配置
  d: 删除配置
```

- **已有配置**: 输入编号直接加载，所有设置（截图区域、模板集、按键等）自动恢复
- **新建配置**: 走完整设置流程
- **删除配置**: 可选择同时删除关联的模板集

---

### 2. 截图区域设置

```
截图区域设置:
1. 自动查找游戏窗口
2. 鼠标拖拽选择区域
3. 使用全屏截图
```

| 选项 | 说明 | 推荐场景 |
|------|------|----------|
| 1. 自动查找 | EnumWindows API匹配窗口类名/标题 | 推荐，最方便 |
| 2. 鼠标拖拽 | 截全屏后用鼠标框选区域 | 多显示器或需要精确区域 |
| 3. 全屏截图 | 使用主显示器全屏 | 游戏全屏时使用 |

自动查找支持的窗口标识：
- 类名：`MapleStoryClass`、`MapleStory`
- 标题：`MapleStory`、`冒险岛`、`新枫之谷`

---

### 3. 检测模式

#### 颜色模式 (`c`)
- 即开即用，无需截图
- 支持红/蓝/绿/黄四种颜色（完整版可选）
- 简化版默认检测红色系怪物
- 可能有误识别（同颜色技能特效等）

#### 模板模式 (`t`)
- 框选怪物图像作为模板，识别更精确
- 支持**模板集**管理：每个地图/怪物的模板保存在独立文件夹
- 可加载已有模板集或新建
- 支持多缩放比例匹配（0.85x / 1.0x / 1.15x）

**模板框选方法**：
1. 脚本截取当前画面
2. 弹出 OpenCV 窗口
3. 用鼠标拖拽框选怪物区域
4. 按空格/回车确认，C取消
5. 可添加多个模板提高识别率

---

### 4. 职业类型

```
职业类型 (m=近战/r=远程): m
```

| 类型 | 攻击距离 | 行为 |
|------|----------|------|
| 近战 (`m`) | 默认80px，可自定义 | 贴脸怪物攻击 |
| 远程 (`r`) | 默认250px，可自定义 | 保持距离攻击 |

### 5. 按键设置

```
跳跃键 (默认space): c
攻击键 (默认x): v
技能键 (逗号分隔, 默认a,s): a,s,d,f,t
是否有位移技能? (y/n, 默认n): y
位移技能键 (默认f): shift
```

所有按键均可自定义，支持字母键和特殊键（space、shift、ctrl等）。

---

### 6. 战斗策略

脚本采用**高度分层决策**：

| 优先级 | 条件 | 动作 |
|--------|------|------|
| 1 | 同层有怪 | BFS聚类找最密集簇，攻击或移向簇中心 |
| 2 | 下方有怪 | 下落穿台（下+跳） |
| 3 | 上方有怪 | 上跳攻击（方向+上+二段跳+攻击） |

附加机制：
- **撞墙检测**: 最近10次移动方向一致时自动换向
- **随机攻击**: 50%概率使用技能，50%概率普攻
- **位移技能**: 配置后可在移动中使用

```
命令 (s=开始/q=退出): s
```

## 高级配置

### 调整检测颜色（简化版）

修改 `mxd_vision_simple.py` 中的颜色范围：

```python
def _detect_color(self, img):
    hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
    # 绿色怪物
    mask = cv2.inRange(hsv, np.array([40, 100, 100]), np.array([80, 255, 255]))
```

常见颜色HSV范围：
| 颜色 | H范围 | 示例怪物 |
|------|-------|----------|
| 红色 | 0-10, 160-180 | 蜗牛、绿水灵 |
| 绿色 | 40-80 | 绿蘑菇 |
| 蓝色 | 100-130 | 蓝水灵 |
| 黄色 | 20-35 | 黄蘑菇 |

### 调整颜色（完整版）

完整版支持交互选择颜色，并可调整最小面积等参数。

### 调整帧率

```python
self.target_fps = 30  # 完整版
```

## 异常处理

### 卡死检测
- **触发条件**: 30帧内无攻击且移动少于3次
- **处理方式**: 连续跳跃+随机方向移动

### 无怪探索
- **触发条件**: 连续5秒无怪物（简化版）或80%时间idle（完整版）
- **处理方式**: 左右移动探索

### 按键保护
- 使用 `try/finally` 保证所有按键操作后必定释放
- 停止时自动调用 `release_all_keys()` 释放所有可能卡住的键

## 故障排除

### 问题1: 识别不到怪物

**解决**:
1. 确认游戏窗口可见，没有被遮挡
2. 尝试模板匹配模式，手动框选怪物
3. 调整颜色检测的 HSV 范围
4. 使用完整版开启调试显示查看检测结果

### 问题2: 按键无反应

**解决**:
1. 确认脚本成功绑定游戏窗口句柄（查看 `[窗口]` 日志）
2. 如果游戏以管理员权限运行，脚本也需要管理员权限
3. 尝试鼠标拖拽方式选择截图区域

### 问题3: 按键卡住

**解决**:
1. 脚本已内置 try/finally 保护，正常不会卡键
2. 如出现卡键，按 ESC 停止脚本，会自动释放所有键
3. 实在无法释放，用 Ctrl+C 退出

### 问题4: 窗口找不到

**解决**:
1. 确保游戏窗口处于可见状态（非最小化）
2. 脚本支持自动匹配 `MapleStoryClass` 等窗口类名
3. 也可手动输入窗口标题或编号选择

## 使用技巧

1. **保存配置**: 首次设置后保存配置，下次直接加载，无需重复设置
2. **模板集**: 同一种怪物只需截图一次，保存为模板集后可反复使用
3. **选择挂机地点**: 怪物刷新快、密度高、地形平坦的小地图效果最好
4. **多配置切换**: 不同角色/地图保存不同配置，一键切换
