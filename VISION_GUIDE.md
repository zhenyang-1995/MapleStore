## 🚀 快速上手

确保已安装依赖：

### 第二步：选择版本

| 版本 | 适用场景 | 推荐度 |
|------|----------|--------|
| `mxd_vision_simple.py` | 快速开始，配置简单 | ⭐⭐⭐ |

## 🎮 使用步骤详解

### 1. 选择检测模式

```
检测模式 (c=颜色/t=模板): 
```

#### 颜色模式 (`c`)
- **适用场景**: 怪物颜色明显且统一（如红色蜗牛、绿水灵）
- **优点**: 无需截图，即开即用
- **缺点**: 可能误识别同颜色物体

#### 模板模式 (`t`)
- **适用场景**: 怪物颜色复杂，需要精确识别
- **优点**: 识别准确
- **缺点**: 需要为每种怪物截图

**模板截图方法**：
1. 进入游戏，走到怪物身边
2. 脚本提示 `对准怪物后按回车截图`
3. 按回车，中央80x80像素区域会被保存为模板
4. 可添加多个模板提高识别率

---

### 2. 选择职业类型

```
职业类型 (m=近战/r=远程): 
```

#### 近战 (`m`)
- 攻击距离：80像素
- 行为：走到怪物身边攻击

#### 远程 (`r`)
- 攻击距离：250像素
- 行为：保持距离攻击

---

### 3. 开始自动刷怪

```
命令 (s=开始/q=退出): s
3秒后开始...
3
2
1
运行中...
```

此时脚本会自动：
1. 截图分析
2. 检测怪物位置
3. 移动或攻击
4. 循环执行

按 `ESC` 停止。

## 🔧 高级配置

### 调整检测颜色

如果怪物不是红色，修改 `mxd_vision_simple.py`：

```python
def _detect_color(self, img):
    hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
    
    # 改为检测绿色怪物（如绿蘑菇）
    mask = cv2.inRange(hsv, np.array([40, 100, 100]), np.array([80, 255, 255]))
    
    # 或检测蓝色怪物
    # mask = cv2.inRange(hsv, np.array([100, 100, 100]), np.array([130, 255, 255]))
```

常见颜色HSV范围：
| 颜色 | H范围 | 示例怪物 |
|------|-------|----------|
| 红色 | 0-10, 160-180 | 蜗牛、绿水灵 |
| 绿色 | 40-80 | 绿蘑菇 |
| 蓝色 | 100-130 | 蓝水灵 |
| 黄色 | 20-35 | 黄蘑菇 |

## 🛡️ 异常处理说明

脚本内置以下异常处理：

### 1. 卡死检测
- **触发条件**: 连续15帧位置变化小于15像素
- **处理方式**: 自动跳跃并左右移动尝试脱困

### 2. 无怪检测
- **触发条件**: 连续5秒没有找到怪物
- **处理方式**: 左右移动探索

### 3. 被击退检测
- **触发条件**: 位置突然大幅变化
- **处理方式**: 短暂停顿后继续

## 🐛 故障排除

### 问题1: 识别不到怪物

**现象**: 脚本运行但没有反应

**解决**:
1. 确认游戏窗口可见，没有被遮挡
2. 尝试模板匹配模式，手动截图
3. 检查怪物颜色，调整HSV范围
4. 开启调试模式查看检测结果

---

### 问题2: 误识别严重

**现象**: 把背景/特效当成怪物

**解决**:
1. 使用模板匹配模式替代颜色模式
2. 调整 `min_contour_area` 过滤小区域
3. 选择特征更明显的怪物

---

### 问题3: 移动不流畅

**现象**: 角色移动卡顿

**解决**:
1. 降低帧率，减少CPU占用
2. 关闭其他占用资源的程序
3. 调整按键持续时间

---

### 问题4: 无法停止脚本

**现象**: 按ESC没有反应

**解决**:
1. 确保游戏窗口不是全屏（窗口化/无边框）
2. 使用 Alt+Tab 切出游戏，在控制台按 Ctrl+C
3. 实在不行使用任务管理器结束Python进程

## 💡 使用技巧

### 技巧1: 选择挂机地点
- 选择怪物刷新快、密度高的地图
- 小地图优先（大地图移动逻辑需要改进）
- 避免有太多障碍物或平台的地形

### 技巧2: 提高识别率
- 保持游戏画面清晰，不要开太多特效
- 怪物在画面中不要过小
- 避免背景颜色和怪物颜色相近

### 技巧3: 安全挂机
- 首次使用先观察10-15分钟
- 不要长时间无人看管
- 定期检查运行状态
